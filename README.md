# YELL VOTE デザイン一新プロトタイプ

YELL VOTE のデザイン刷新プロトタイプ（単一HTML / バニラJS）。

## 構成
- `prototype.html` — インタラクティブ・プロトタイプ本体
- `img/` — ロゴ素材（SVG）

## 使い方
ローカルで `prototype.html` を直接開くか、簡易サーバーで配信します。

```bash
npx serve . -l 4910
# → http://localhost:4910/prototype.html
```

## 主な画面
- TOP（広告枠 / 開催中オーディション / NEWS）
- オーディション一覧
- オーディション詳細（TOP・概要のタブ／**通常パターン** と **ミスコンパターン** の2種）
- エントリー詳細（順位・獲得ポイント・写真・インタビュー・応援コメント）
- 結果発表 / マイページ / ポイント購入 / YELL VOTEとは / よくある質問 / NEWS / 法務

## メモ
- レスポンシブ対応（SP / タブレット / PC）
- 投票・コメントはログイン前提（ヘッダーの「ログイン/ログアウト」で挙動を確認できます）

> 本リポジトリはデザイン検証用のプロトタイプです。掲載データはすべてサンプルです。
